# Code of Conduct

We follow the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**Short version:**

- Be kind. Assume good faith.
- No harassment, discrimination, or personal attacks.
- Disagreement is fine; contempt is not.
- This is a security project. We are even more careful than usual about not making people feel stupid for not knowing something — most users of this tool were tricked into installing the thing it removes, and we never make them feel bad about that.
- Maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that don't align with this Code of Conduct.

**Enforcement:** report incidents to the project maintainers (contact info in the [README](README.md)). All reports will be reviewed and investigated promptly and fairly. Reporter identity will be kept confidential.

**Scope:** this Code of Conduct applies in all project spaces (GitHub, any chat channels we set up, any in-person events) and when an individual is representing the project in public.
